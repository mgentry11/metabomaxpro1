Essential Concepts

What is Multivariable Calculus?

In Calculus I and II, you worked with functions of one variable: y = f(x). In Calculus III, we extend these ideas to functions of multiple variables: z = f(x, y) or w = f(x, y, z). This lets us work with surfaces, volumes, and fields in 2D and 3D space.

The Four Types of Integrals

What We're Computing

  • Double Integrals ∬ - Sum over a 2D region (area, mass of a plate)
  • Triple Integrals ∭ - Sum over a 3D region (volume, mass of a solid)
  • Line Integrals \int - Sum along a curve (work done by a force)
  • Surface Integrals ∬ - Sum over a surface (flux through a membrane)

Coordinate Systems

Polar Coordinates (2D)
x = r cos \theta, y = r sin \theta, r^2 = x^2 + y^2 dA = r \, dr \, d\theta ← Don't forget the r!
Cylindrical Coordinates (3D)
x = r cos \theta, y = r sin \theta, z = z dV = r dz \, dr \, d\theta
Spherical Coordinates (3D)
x = \rho sin \phi cos \theta, y = \rho sin \phi sin \theta, z = \rho cos \phi \rho^2 = x^2 + y^2 + z^2 dV = \rho^2 sin \phi \, d\rho \, d\phi d\theta
Use polar/cylindrical when you see circles (x^2 + y^2). Use spherical when you see spheres (x^2 + y^2 + z^2).

Problem 2 Iterated Integral of Cosine

Evaluate: $\int_{-\pi}^{0} \int_{\pi/4}^{\pi/2} \cos(2x + y) \, dy \, dx$

Answer: 0

Solution
Inner integral (with respect to y)
$$\int _{\pi/4}^{\pi/2} \cos(2x + y) dy = [\sin(2x + y)]_{\pi/4}^{\pi/2} = \sin(2x + \pi/2) - \sin(2x + \pi/4) = \cos(2x) - \sin(2x + \pi/4)
Outer integral (with respect to x)
$$\int _{-\pi}^{0} [\cos(2x) - \sin(2x + \pi/4)] dx = [\sin(2x)/2 + \cos(2x + \pi/4)/2]_{-\pi}^{0}
Evaluate at bounds
$$At x = 0: \sin(0)/2 + \cos(\pi/4)/2 = 0 + (\sqrt{2}/2)/2 = \sqrt{2}/4 At x = -\pi: \sin(-2\pi)/2 + \cos(-2\pi + \pi/4)/2 = 0 + \cos(\pi/4)/2 = \sqrt{2}/4 Result: \sqrt{2}/4 - \sqrt{2}/4 = 0
Why Zero?

The integral is zero because we're integrating periodic functions over complete periods, causing positive and negative contributions to cancel.

ANSWER
0

Problem 4 Question 16.138: Gaussian Over Lower Semicircle

Evaluate \iint_B e^{-x^2-y^2} \, dx \, dy, where B = {(x,y) | x^2 + y^2 \leq 1 and y \leq 0}

Solution
Convert to polar coordinates
$$x^2 + y^2 = r^2, so e^{-x^2-y^2} = e^{-r^2} dA = r \, dr \, d\theta Region: 0 \leq r \leq 1, \pi \leq \theta \leq 2\pi (lower half where y \leq 0)
Set up integral
$$\iint_B e^{-r^2} \cdot r \, dr \, d\theta = \int_{\pi}^{2\pi} \int_0^1 r \cdot e^{-r^2} \, dr \, d\theta$$
Inner integral
$$\int_0^1 r \cdot e^{-r^2} dr Let u = -r^2, du = -2r dr, so r dr = -du/2 = -½[e^u]_{0}^{-1} = -½(e^{-1} - 1) = ½(1 - 1/e) = (e-1)/(2e)
Outer integral
$$\int_{\pi}^{2\pi} (e-1)/(2e) d\theta = (e-1)/(2e) \cdot \pi = \pi(e-1)/(2e)$$
ANSWER
\pi(1 - e^{-1})/2 = \pi(e-1)/(2e)
Problem 5 Question 16.147b: Fourth Power Integral

Evaluate \iint_B (x^4 + 2x^2y^2 + y^4) \, dx \, dy where B is the disk of radius 2 centered at origin in the first quadrant.

Solution
Recognize the pattern
$$x^4 + 2x^2y^2 + y^4 = (x^2 + y^2)^2 = r^4 in polar$$
Convert to polar
$$\iint_B r^4 \cdot r \, dr \, d\theta = \int_0^{\pi/2} \int_0^2 r^5 \, dr \, d\theta$$
Evaluate
$$\int_0^2 r^5 dr = [r^6/6]_0^2 = 64/6 = 32/3 \int_0^{\pi/2} (32/3) d\theta = (32/3)(\pi/2) = 16\pi/3
ANSWER
16\pi/3
Problem 6 Question 16.147a: Improper Double Integral

Evaluate: \int_0^{\infty} \int_0^y x \cdot e^{-y^3} \, dx \, dy

Solution
Inner integral (with respect to x)
$$\int_0^y x \cdot e^{-y^3} dx = e^{-y^3} \int_0^y x dx = e^{-y^3} \cdot [x^2/2]_0^y = e^{-y^3} \cdot y^2/2$$
Outer integral
$$\int_0^{\infty} (y^2/2) \cdot e^{-y^3} dy Let u = y^3, du = 3y^2 dy, so y^2 dy = du/3 = (1/2) \cdot (1/3) \int_0^{\infty} e^{-u} du = (1/6)[-e^{-u}]_0^\infty = (1/6)(0 - (-1)) = 1/6
ANSWER
1/6
Problem 7 Question 16.149: Integral Over All of R^2

Evaluate ∬_{R^2} f(x,y) \, dx \, dy where f(x,y) = 1/(1 + x^2 + y^2)^{3/2}

Solution
Convert to polar
$$1 + x^2 + y^2 = 1 + r^2 f = 1/(1 + r^2)^{3/2} dA = r \, dr \, d\theta \int_0^{2\pi} \int_0^{\infty} r/(1 + r^2)^{3/2} \, dr \, d\theta

Problem 8 Integral with Square Root

Evaluate \iint_D 1/\sqrt{(3xy) dA where D = [0,8] \times [0,8]

Solution
Separate the integral
$$\iint_D 1/\sqrt{(3xy) dA = (1/\sqrt{3}) \int_0^8 \int_0^8 x^{-1/2} y^{-1/2} \, dx \, dy = (1/\sqrt{3}) \int_0^8 x^{-1/2} dx \cdot \int_0^8 y^{-1/2} dy
Evaluate each integral
$$\int_0^8 x^{-1/2} dx = [2x^{1/2}]_0^8 = 2\sqrt{8} = 4\sqrt{2 Similarly for y: 4\sqrt{2 Result: (1/\sqrt{3})(4\sqrt{2)(4\sqrt{2) = (1/\sqrt{3})(32) = 32/\sqrt{3} = 32\sqrt{3}/3
This integral is improper at x=0 and y=0, but converges because the exponent -1/2 > -1.
ANSWER
32\sqrt{3}/3

Problem 9 Triple Integral with Symmetry

Evaluate ∭_W 9x dV for region W: x^2 + y^2 \leq z \leq 81

Answer: 0

Solution
Identify symmetry

The region W is symmetric about the yz-plane (if you replace x with -x, you get the same region).

The function f(x,y,z) = 9x is odd in x: f(-x,y,z) = -9x = -f(x,y,z)

Apply symmetry principle

When integrating an odd function over a symmetric region, the integral is zero.

The Symmetry Shortcut

For every point (x,y,z) contributing +9x, there's a point (-x,y,z) contributing -9x. They cancel!

ANSWER
0
Problem 10 Improper Integral Outside Sphere

Let D be the unbounded region x^2 + y^2 + z^2 \geq 9. Evaluate: ∭_D 7/(x^2 + y^2 + z^2)^2 dV

Answer: 28\pi/3

Solution
Convert to spherical
$$x^2 + y^2 + z^2 = \rho^2, so 7/(x^2 + y^2 + z^2)^2 = 7/\rho^4 dV = \rho^2 sin \phi \, d\rho \, d\phi d\theta Region: \rho \geq 3, 0 \leq \phi \leq \pi, 0 \leq \theta \leq 2\pi
Set up integral
$$\int_0^{2\pi} \int_0^{\pi} \int_3^{\infty} (7/\rho^4) \cdot \rho^2 sin \phi \, d\rho \, d\phi d\theta = \int_0^{2\pi} \int_0^{\pi} \int_3^{\infty} 7\rho^{-2} sin \phi \, d\rho \, d\phi d\theta
Evaluate \rho integral
$$\int_3^{\infty} 7\rho^{-2} d\rho = 7[-\rho^{-1}]_3^\infty = 7(0 + 1/3) = 7/3$$
Evaluate \phi integral
$$\int_0^{\pi} sin \phi d\phi = [-cos \phi]_0^\pi = 1 + 1 = 2$$
Evaluate \theta integral
$$\int_0^{2\pi} d\theta = 2\pi$$
Combine
$$(7/3) \times 2 \times 2\pi = 28\pi/3$$
ANSWER
28\pi/3

Problem 12 Question 16.140: Triple Integral Over Sphere/Hemisphere/Octant

Evaluate ∭_D (x^2 + y^2 + z^2)xyz dV over:

a) The sphere D = {x^2 + y^2 + z^2 \leq R^2}

b) The hemisphere D = {x^2 + y^2 + z^2 \leq R^2, z \geq 0}

Part (a): Full sphere
Symmetry argument

The integrand xyz is odd in each variable. The sphere is symmetric about each coordinate plane.

$$Answer (a): 0$$

$$Answer (b): 0$$

Problem 13 Question 16.141: Cone-Shaped Region

Let C be the cone-shaped region {(x,y,z) | \sqrt{(x^2 + y^2) \leq z \leq 1}. Evaluate ∭_C (1 + \sqrt{(x^2 + y^2)) dV

Solution
Convert to cylindrical
$$\sqrt{(x^2 + y^2) = r, so region is: r \leq z \leq 1, 0 \leq r \leq 1 Integrand: 1 + r dV = r dz \, dr \, d\theta
Set up integral
$$\int_0^{2\pi} \int_0^1 \int_r^1 (1 + r) \cdot r dz \, dr \, d\theta$$
Inner integral (z)
$$\int_r^1 dz = 1 - r$$
Middle integral (r)
$$\int_0^1 (1 + r) \cdot r \cdot (1 - r) dr = \int_0^1 (r + r^2 - r^2 - r^3) dr = \int_0^1 (r - r^3) dr = [r^2/2 - r^4/4]_0^1 = 1/2 - 1/4 = 1/4
Outer integral (\theta)
$$\int_0^{2\pi} (1/4) d\theta = \pi/2$$
ANSWER
\pi/2

Green's Theorem
∮_C (P dx + Q dy) = \iint_D (\partialQ/\partialx - \partialP/\partialy) dA
Problem 33 Green's Theorem Application

Use Green's Theorem to evaluate ∮_C F \cdot dr where F(x,y) = (x^2, x^2), C consists of y = x^2 and y = 4x for 0 \leq x \leq 4, counterclockwise.

Solution
Identify P and Q
$$P = x^2, Q = x^2 \partialQ/\partialx = 2x, \partialP/\partialy = 0
Apply Green's Theorem
$$∮_C F \cdot dr = \iint_D (2x - 0) dA = \iint_D 2x dA$$
Evaluate double integral
$$\int_0^4 \int_{x^2}^{4x} 2x \, dy \, dx = \int_0^4 2x(4x - x^2) dx = \int_0^4 (8x^2 - 2x^3) dx = [8x^3/3 - x^4/2]_0^4 = 512/3 - 128 = 128/3
ANSWER
128/3

Test for Conservative (2D)
F = ⟨P, Q⟩ is conservative ⟺ \partialP/\partialy = \partialQ/\partialx

Find f

f(x,y) = (3/2)x^2 + (7/2)y^2 + C

Problem 42 Conservative Field (c): F = (7x^2 + 7y^2)i + 14xyj
