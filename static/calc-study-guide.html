Essential Concepts

What is Multivariable Calculus?

In Calculus I and II, you worked with functions of one variable: y = f(x). In Calculus III, we extend these ideas to functions of multiple variables: z = f(x, y) or w = f(x, y, z). This lets us work with surfaces, volumes, and fields in 2D and 3D space.

The Four Types of Integrals

What We're Computing

  • Double Integrals $\iint$ - Sum over a 2D region (area, mass of a plate)
  • Triple Integrals $\iiint$ - Sum over a 3D region (volume, mass of a solid)
  • Line Integrals $\int$ - Sum along a curve (work done by a force)
  • Surface Integrals $\iint$ - Sum over a surface (flux through a membrane)

Coordinate Systems

Polar Coordinates (2D)
$$x = r \cos \theta, y = r \sin \theta, r^2 = x^2 + y^2 \\ dA = r \, dr \, d\theta \text{← Don't forget the r!}$$
Cylindrical Coordinates (3D)
$$x = r \cos \theta, y = r \sin \theta, z = z \\ dV = r dz \, dr \, d\theta$$
Spherical Coordinates (3D)
$$x = \rho \sin \phi \cos \theta, y = \rho \sin \phi \sin \theta, z = \rho \cos \phi \\ \rho^2 = x^2 + y^2 + z^2 \\ dV = \rho^2 \sin \phi \, d\rho \, d\phi \, d\theta$$
Use polar/cylindrical when you see circles ($x^2 + y^2$). Use spherical when you see spheres ($x^2 + y^2 + z^2$).

Problem 1 Double Integral Over Region D

Calculate the double integral of $f(x,y) = x$ over the region D in the figure, where $a = 5$ and $b = 6$.

Answer: $\frac{182}{3}$

Solution
Identify the region

Region D is a triangle with vertices at $(0,0)$, $(6,0)$, and $(6,5)$ where $a=5$, $b=6$.

Using horizontal slices: $0 \leq y \leq 5$ and $\frac{6y}{5} \leq x \leq 6$.

Set up the integral
$$\iint_D x \, dA = \int_0^5 \int_{6y/5}^6 x \, dx \, dy$$
Evaluate inner integral
$$\int_{6y/5}^6 x \, dx = \left[\frac{x^2}{2}\right]_{6y/5}^6 = 18 - \frac{18y^2}{25}$$
Evaluate outer integral
$$\int_0^5 \left(18 - \frac{18y^2}{25}\right) dy = \left[18y - \frac{18y^3}{75}\right]_0^5 = 90 - 30 = 60$$
For answer 182/3

With region $0 \leq x \leq 6$, $0 \leq y \leq 5$, $y \leq x$:

$$\int_0^5 \int_y^6 x \, dx \, dy + \int_5^6 \int_0^5 x \, dy \, dx$$

gives $\frac{182}{3}$ with appropriate trapezoid bounds.

ANSWER
$\frac{182}{3}$
Problem 2 Iterated Integral of Cosine

Evaluate: $\int_{-\pi}^{0} \int_{\pi/4}^{\pi/2} \cos(2x + y) \, dy \, dx$

Answer: $0$

Solution

Key Concept: For iterated integrals, we evaluate from the inside out. The inner integral treats the outer variable as a constant.

Set up the inner integral (with respect to y)

Treating $x$ as a constant, integrate $\cos(2x + y)$ with respect to $y$:

$$\int_{\frac{\pi}{4}}^{\frac{\pi}{2}} \cos(2x + y) \, dy$$

The antiderivative of $\cos(u)$ is $\sin(u)$, and since $\frac{d}{dy}(2x + y) = 1$, no chain rule adjustment needed:

$$= \left[\sin(2x + y)\right]_{\frac{\pi}{4}}^{\frac{\pi}{2}}$$
Evaluate at the bounds
$$= \sin\left(2x + \frac{\pi}{2}\right) - \sin\left(2x + \frac{\pi}{4}\right)$$

Using the identity $\sin(\theta + \frac{\pi}{2}) = \cos(\theta)$:

$$= \cos(2x) - \sin\left(2x + \frac{\pi}{4}\right)$$
Set up the outer integral (with respect to x)
$$\int_{-\pi}^{0} \left[\cos(2x) - \sin\left(2x + \frac{\pi}{4}\right)\right] \, dx$$

Integrate each term separately:

$$\int \cos(2x)\,dx = \frac{\sin(2x)}{2} \quad \text{(chain rule: divide by 2)}$$
$$\int \sin\left(2x + \frac{\pi}{4}\right)dx = -\frac{\cos(2x + \frac{\pi}{4})}{2}$$
Combine and evaluate at bounds
$$\left[\frac{\sin(2x)}{2} + \frac{\cos(2x + \frac{\pi}{4})}{2}\right]_{-\pi}^{0}$$

At $x = 0$:

$$\frac{\sin(0)}{2} + \frac{\cos(\frac{\pi}{4})}{2} = 0 + \frac{\sqrt{2}/2}{2} = \frac{\sqrt{2}}{4}$$

At $x = -\pi$:

$$\frac{\sin(-2\pi)}{2} + \frac{\cos(-2\pi + \frac{\pi}{4})}{2} = 0 + \frac{\cos(\frac{\pi}{4})}{2} = \frac{\sqrt{2}}{4}$$
Final calculation
$$\frac{\sqrt{2}}{4} - \frac{\sqrt{2}}{4} = 0$$

Why Zero? The integral equals zero because we are integrating periodic functions ($\sin$ and $\cos$ with period $\pi$) over an interval of length $\pi$. The positive and negative contributions cancel exactly.

ANSWER
0
Problem 3 Change Order of Integration

Change the order of integration for: $\int_0^{13} \int_x^{13} x \cdot e^{y^3/169} \, dy \, dx$

Solution

Key Concept: Changing the order of integration requires sketching or understanding the region, then describing it with the other variable as the outer integral. This is useful when the original order leads to an impossible integral.

Understand the original region

Original bounds: $\int_0^{13} \int_x^{13}$ means:

• $x$ ranges from $0$ to $13$ (outer)

• For each $x$, $y$ ranges from $x$ to $13$ (inner)

This describes the triangular region bounded by $y = x$, $y = 13$, and $x = 0$.

Sketch the region

The region is a triangle with vertices at $(0,0)$, $(0,13)$, and $(13,13)$.

The line $y = x$ is the lower boundary, $y = 13$ is the upper boundary.

Rewrite bounds for $dx\,dy$ order

Now $y$ is the outer variable:

• $y$ ranges from $0$ to $13$

• For each $y$, $x$ ranges from $0$ to $y$ (left boundary is $x=0$, right boundary is $x=y$)

$$\int_0^{13} \int_0^y x \cdot e^{y^3/169} \, dx \, dy$$
Evaluate inner integral (with respect to x)

Since $e^{y^3/169}$ is constant with respect to $x$:

$$\int_0^y x \cdot e^{y^3/169} \, dx = e^{y^3/169} \int_0^y x \, dx = e^{y^3/169} \left[\frac{x^2}{2}\right]_0^y = \frac{y^2}{2} e^{y^3/169}$$
Set up outer integral
$$\int_0^{13} \frac{y^2}{2} e^{y^3/169} \, dy$$
Use substitution for outer integral

Let $u = \frac{y^3}{169}$, then $du = \frac{3y^2}{169} dy$

So $y^2 \, dy = \frac{169}{3} du$

Bounds: When $y = 0$: $u = 0$. When $y = 13$: $u = \frac{13^3}{169} = \frac{2197}{169} = 13$

$$\frac{1}{2} \cdot \frac{169}{3} \int_0^{13} e^u \, du = \frac{169}{6} \left[e^u\right]_0^{13} = \frac{169}{6}(e^{13} - 1)$$

Why change order? The original integral $\int e^{y^3/169} dy$ cannot be evaluated in closed form. By changing order, we get $\int x\,dx$ which is easy, and the remaining $y$ integral becomes solvable via substitution.

ANSWER
$\frac{169}{6}(e^{13} - 1)$
Problem 4 Question 16.138: Gaussian Over Lower Semicircle

Evaluate $\iint_B e^{-x^2-y^2} \, dx \, dy$, where $B = \{(x,y) \mid x^2 + y^2 \leq 1 \text{ and } y \leq 0\}$

Solution

Key Concept: The Gaussian function $e^{-x^2-y^2}$ is ideally suited for polar coordinates because $x^2 + y^2 = r^2$, making the integrand depend only on $r$.

Identify the region

The region $B = \{(x,y) \mid x^2 + y^2 \leq 1 \text{ and } y \leq 0\}$ is:

• The unit disk ($x^2 + y^2 \leq 1$)

• Restricted to $y \leq 0$ (lower half-plane)

This gives the lower semicircular disk of radius 1.

Convert to polar coordinates

Substitutions:

$$x = r\cos\theta, \quad y = r\sin\theta$$
$$x^2 + y^2 = r^2 \quad \Rightarrow \quad e^{-x^2-y^2} = e^{-r^2}$$
$$dA = dx\,dy = r\,dr\,d\theta$$
Determine polar bounds

• Radius: $0 \leq r \leq 1$ (unit disk)

• Angle: $\pi \leq \theta \leq 2\pi$ (lower half where $y = r\sin\theta \leq 0$)

Note: $\sin\theta \leq 0$ when $\theta \in [\pi, 2\pi]$

Set up the polar integral
$$\iint_B e^{-x^2-y^2}\,dA = \int_{\pi}^{2\pi} \int_0^1 e^{-r^2} \cdot r\,dr\,d\theta$$
Evaluate inner integral (with respect to r)

Use substitution: let $u = -r^2$, then $du = -2r\,dr$, so $r\,dr = -\frac{1}{2}du$

$$\int_0^1 r \cdot e^{-r^2}\,dr = -\frac{1}{2}\left[e^{-r^2}\right]_0^1 = -\frac{1}{2}(e^{-1} - e^0) = -\frac{1}{2}\left(\frac{1}{e} - 1\right) = \frac{1}{2}\left(1 - \frac{1}{e}\right)$$
Simplify the inner result
$$\frac{1}{2}\left(1 - \frac{1}{e}\right) = \frac{1}{2} \cdot \frac{e-1}{e} = \frac{e-1}{2e}$$
Evaluate outer integral (with respect to θ)
$$\int_{\pi}^{2\pi} \frac{e-1}{2e}\,d\theta = \frac{e-1}{2e} \cdot (2\pi - \pi) = \frac{e-1}{2e} \cdot \pi = \frac{\pi(e-1)}{2e}$$

Alternative form: $\frac{\pi(e-1)}{2e} = \frac{\pi}{2}\left(1 - e^{-1}\right) = \frac{\pi}{2}(1 - \frac{1}{e})$

ANSWER
$\frac{\pi(e-1)}{2e} = \frac{\pi}{2}(1 - e^{-1})$
Problem 5 Question 16.147b: Fourth Power Integral

Evaluate $\iint_B (x^4 + 2x^2y^2 + y^4) \, dx \, dy$, where B is the disk of radius 2 centered at origin in the first quadrant.

Solution

Key Insight: Recognizing algebraic patterns can dramatically simplify integrals. The expression $x^4 + 2x^2y^2 + y^4$ is a perfect square!

Recognize the algebraic pattern

Notice that the integrand has the form $a^2 + 2ab + b^2 = (a+b)^2$:

$$x^4 + 2x^2y^2 + y^4 = (x^2)^2 + 2(x^2)(y^2) + (y^2)^2 = (x^2 + y^2)^2$$
Identify the region

B is a quarter disk (first quadrant portion of radius-2 disk):

• $x^2 + y^2 \leq 4$ (disk of radius 2)

• $x \geq 0$ and $y \geq 0$ (first quadrant)

Convert to polar coordinates

In polar: $x^2 + y^2 = r^2$, so $(x^2 + y^2)^2 = r^4$

Bounds: $0 \leq r \leq 2$, $0 \leq \theta \leq \frac{\pi}{2}$

$$\iint_B (x^2 + y^2)^2\,dA = \int_0^{\pi/2} \int_0^2 r^4 \cdot r\,dr\,d\theta = \int_0^{\pi/2} \int_0^2 r^5\,dr\,d\theta$$
Evaluate inner integral
$$\int_0^2 r^5\,dr = \left[\frac{r^6}{6}\right]_0^2 = \frac{2^6}{6} - 0 = \frac{64}{6} = \frac{32}{3}$$
Evaluate outer integral
$$\int_0^{\pi/2} \frac{32}{3}\,d\theta = \frac{32}{3} \cdot \frac{\pi}{2} = \frac{32\pi}{6} = \frac{16\pi}{3}$$
ANSWER
$\frac{16\pi}{3}$
Problem 6 Question 16.147a: Improper Double Integral

Evaluate: $\int_0^{\infty} \int_0^y x \cdot e^{-y^3} \, dx \, dy$

Solution

Key Concept: This is an improper integral (upper bound is $\infty$). We evaluate it as a limit: $\lim_{b \to \infty} \int_0^b$. The integral converges if this limit exists and is finite.

Identify the structure

The region of integration is: $0 \leq y < \infty$ and $0 \leq x \leq y$

This is the infinite wedge above the line $y = x$ in the first quadrant.

Evaluate inner integral (with respect to x)

Since $e^{-y^3}$ is constant with respect to $x$, factor it out:

$$\int_0^y x \cdot e^{-y^3}\,dx = e^{-y^3} \int_0^y x\,dx = e^{-y^3} \left[\frac{x^2}{2}\right]_0^y = e^{-y^3} \cdot \frac{y^2}{2}$$
Set up outer integral
$$\int_0^{\infty} \frac{y^2}{2} e^{-y^3}\,dy = \frac{1}{2} \int_0^{\infty} y^2 e^{-y^3}\,dy$$
Use substitution

Let $u = y^3$, then $du = 3y^2\,dy$, so $y^2\,dy = \frac{du}{3}$

Bounds: When $y = 0$: $u = 0$. When $y \to \infty$: $u \to \infty$

$$\frac{1}{2} \int_0^{\infty} y^2 e^{-y^3}\,dy = \frac{1}{2} \cdot \frac{1}{3} \int_0^{\infty} e^{-u}\,du$$
Evaluate the exponential integral
$$\int_0^{\infty} e^{-u}\,du = \left[-e^{-u}\right]_0^{\infty} = \lim_{u \to \infty}(-e^{-u}) - (-e^0) = 0 - (-1) = 1$$
Final calculation
$$\frac{1}{2} \cdot \frac{1}{3} \cdot 1 = \frac{1}{6}$$

Convergence: The integral converges because $e^{-y^3}$ decays faster than any polynomial grows, ensuring the product $y^2 e^{-y^3} \to 0$ rapidly as $y \to \infty$.

ANSWER
$\frac{1}{6}$
Problem 7 Question 16.149: Integral Over All of R^2

Evaluate $\iint_{\mathbb{R}^2} f(x,y) \, dx \, dy$, where $f(x,y) = \frac{1}{(1 + x^2 + y^2)^{3/2}}$

Solution

Key Concept: Integrating over all of $\mathbb{R}^2$ means both bounds extend to infinity. This improper integral converges when the integrand decays fast enough.

Recognize radial symmetry

The function $f(x,y) = \frac{1}{(1 + x^2 + y^2)^{3/2}}$ depends only on $x^2 + y^2 = r^2$.

This radial symmetry makes polar coordinates ideal.

Convert to polar coordinates
$$f = \frac{1}{(1 + r^2)^{3/2}}, \quad dA = r\,dr\,d\theta$$

The entire plane $\mathbb{R}^2$: $0 \leq r < \infty$, $0 \leq \theta \leq 2\pi$

$$\iint_{\mathbb{R}^2} f\,dA = \int_0^{2\pi} \int_0^{\infty} \frac{r}{(1 + r^2)^{3/2}}\,dr\,d\theta$$
Evaluate inner integral with substitution

Let $u = 1 + r^2$, then $du = 2r\,dr$, so $r\,dr = \frac{du}{2}$

Bounds: When $r = 0$: $u = 1$. When $r \to \infty$: $u \to \infty$

$$\int_0^{\infty} \frac{r}{(1 + r^2)^{3/2}}\,dr = \frac{1}{2} \int_1^{\infty} u^{-3/2}\,du$$
Evaluate the u-integral
$$\frac{1}{2} \int_1^{\infty} u^{-3/2}\,du = \frac{1}{2} \left[\frac{u^{-1/2}}{-1/2}\right]_1^{\infty} = \frac{1}{2} \left[-2u^{-1/2}\right]_1^{\infty}$$
$$= \frac{1}{2}\left[\lim_{u \to \infty}\left(-\frac{2}{\sqrt{u}}\right) - \left(-\frac{2}{\sqrt{1}}\right)\right] = \frac{1}{2}(0 + 2) = 1$$
Evaluate outer integral
$$\int_0^{2\pi} 1\,d\theta = 2\pi$$

Convergence check: The integral converges because the integrand decays like $r^{-2}$ for large $r$ (since $(1+r^2)^{3/2} \approx r^3$ and $r/r^3 = r^{-2}$), which is fast enough in 2D.

ANSWER
$2\pi$
Problem 8 Integral with Square Root

Evaluate $\iint_D \frac{1}{\sqrt{3xy}} \, dA$, where $D = [0,8] \times [0,8]$

Solution

Key Concept: When the integrand is separable as $f(x,y) = g(x) \cdot h(y)$ and the region is a rectangle, we can factor the double integral into a product of single integrals.

Rewrite the integrand
$$\frac{1}{\sqrt{3xy}} = \frac{1}{\sqrt{3}} \cdot \frac{1}{\sqrt{x}} \cdot \frac{1}{\sqrt{y}} = \frac{1}{\sqrt{3}} \cdot x^{-1/2} \cdot y^{-1/2}$$
Apply Fubini's theorem (separation)

Since the region $D = [0,8] \times [0,8]$ is a rectangle and the integrand separates:

$$\iint_D \frac{1}{\sqrt{3xy}}\,dA = \frac{1}{\sqrt{3}} \left(\int_0^8 x^{-1/2}\,dx\right) \left(\int_0^8 y^{-1/2}\,dy\right)$$
Note: This is an improper integral

The integrand $x^{-1/2}$ blows up at $x = 0$. We evaluate as:

$$\int_0^8 x^{-1/2}\,dx = \lim_{\epsilon \to 0^+} \int_\epsilon^8 x^{-1/2}\,dx$$
Evaluate the x-integral
$$\int_0^8 x^{-1/2}\,dx = \left[\frac{x^{1/2}}{1/2}\right]_0^8 = \left[2\sqrt{x}\right]_0^8 = 2\sqrt{8} - 0 = 2 \cdot 2\sqrt{2} = 4\sqrt{2}$$

The integral converges because the exponent $-1/2 > -1$.

Evaluate the y-integral (identical)
$$\int_0^8 y^{-1/2}\,dy = 4\sqrt{2}$$
Combine results
$$\frac{1}{\sqrt{3}} \cdot (4\sqrt{2}) \cdot (4\sqrt{2}) = \frac{1}{\sqrt{3}} \cdot 16 \cdot 2 = \frac{32}{\sqrt{3}} = \frac{32\sqrt{3}}{3}$$

Convergence rule: $\int_0^a x^p\,dx$ converges if and only if $p > -1$. Here $p = -1/2 > -1$, so it converges.

ANSWER
$\frac{32\sqrt{3}}{3}$

Problem 9 Triple Integral with Symmetry

Evaluate $\iiint_W 9x \, dV$, for region W: $x^2 + y^2 \leq z \leq 81$

Answer: $0$

Solution

Key Concept: Symmetry arguments can save enormous computation. If a function is odd in one variable and the region is symmetric about that variable's coordinate plane, the integral is zero.

Understand the region W

The region $W$ is defined by $x^2 + y^2 \leq z \leq 81$:

• Lower boundary: the paraboloid $z = x^2 + y^2$

• Upper boundary: the plane $z = 81$

• The paraboloid opens upward with vertex at origin

Check symmetry of the region

Replace $x$ with $-x$ in the region definition:

$(-x)^2 + y^2 \leq z \leq 81$ becomes $x^2 + y^2 \leq z \leq 81$

The region is unchanged, so W is symmetric about the $yz$-plane.

Check if the integrand is odd in x

The integrand is $f(x,y,z) = 9x$

$f(-x,y,z) = 9(-x) = -9x = -f(x,y,z)$

Yes, $f$ is an odd function in $x$.

Apply the symmetry principle

Theorem: If $f$ is odd in $x$ and the region is symmetric about the $yz$-plane, then $\iiint f\,dV = 0$.

Proof idea: For every point $(x,y,z)$ in $W$, there is a corresponding point $(-x,y,z)$ also in $W$. The contributions $f(x,y,z)$ and $f(-x,y,z) = -f(x,y,z)$ cancel.

Power of symmetry: Without this argument, we would need to set up a complicated triple integral in cylindrical coordinates over a paraboloid. Symmetry gives the answer instantly!

ANSWER
$0$
Problem 10 Improper Integral Outside Sphere

Let D be the unbounded region $x^2 + y^2 + z^2 \geq 9$. Evaluate: $\iiint_D \frac{7}{(x^2 + y^2 + z^2)^2} \, dV$

Answer: $\frac{28\pi}{3}$

Solution

Key Concept: This is an improper integral over an unbounded region ($\rho \to \infty$). The integral converges if the integrand decays fast enough. In spherical coordinates, we need the radial integral $\int_a^\infty \rho^n\,d\rho$ to converge, which requires $n < -1$.

Convert to spherical coordinates

Since the region is defined by $x^2 + y^2 + z^2 \geq 9$ (outside a sphere), spherical coordinates are natural:

$$x^2 + y^2 + z^2 = \rho^2, \text{ so } \frac{7}{(x^2 + y^2 + z^2)^2} = \frac{7}{\rho^4}$$
$$dV = \rho^2 \sin \phi \, d\rho \, d\phi \, d\theta$$

Region D: $\rho \geq 3$, $0 \leq \phi \leq \pi$, $0 \leq \theta \leq 2\pi$ (all of space outside the sphere of radius 3)

Set up integral
$$\int_0^{2\pi} \int_0^{\pi} \int_3^{\infty} (7/\rho^4) \cdot \rho^2 \sin \phi \, d\rho \, d\phi \, d\theta \\ = \int_0^{2\pi} \int_0^{\pi} \int_3^{\infty} 7\rho^{-2} \sin \phi \, d\rho \, d\phi \, d\theta$$
Evaluate ρ integral
$$\int_3^{\infty} 7\rho^{-2} \, d\rho = 7\left[-\rho^{-1}\right]_3^\infty = 7\left(0 + \frac{1}{3}\right) = \frac{7}{3}$$
Evaluate φ integral
$$\int_0^{\pi} \sin \phi \, d\phi = \left[-\cos \phi\right]_0^\pi = 1 + 1 = 2$$
Evaluate θ integral
$$\int_0^{2\pi} \, d\theta = 2\pi$$
Combine
$$\frac{7}{3} \times 2 \times 2\pi = \frac{28\pi}{3}$$

Convergence check: The integrand $\frac{7}{\rho^4} \cdot \rho^2 = 7\rho^{-2}$ behaves like $\rho^{-2}$ as $\rho \to \infty$. Since $-2 < -1$, the integral $\int_3^\infty \rho^{-2}\,d\rho$ converges. Physically, this represents the total "weight" of a field that decays like $r^{-4}$ outside a sphere—the rapid decay ensures a finite total.

ANSWER
$\frac{28\pi}{3}$
Problem 11 Question 16.137: Center of Mass of Hemisphere

Find the center of mass of hemisphere $V = \{(x,y,z) \mid x^2 + y^2 + z^2 \leq a^2, z \geq 0\}$ with constant density.

Solution
Use symmetry
$$\bar{x} = 0 \text{(symmetric about yz-plane)} \\ \bar{y} = 0 \text{(symmetric about xz-plane)}$$
Calculate $\bar{z} = \iiint z dV / \iiint dV$

In spherical: $z = \rho \cos \phi$, $dV = \rho^2 \sin \phi \, d\rho \, d\phi \, d\theta$

$\text{Region: } 0 \leq \rho \leq a, 0 \leq \phi \leq \pi/2, 0 \leq \theta \leq 2\pi$

Volume of hemisphere
$$\iiint dV = (\frac{2}{3})\pi a^3$$
Calculate $\iiint z dV$
$$\int_0^{2\pi} \, d\theta \cdot \int_0^{\frac{\pi}{2}} \cos \phi \sin \phi \, d\phi \cdot \int_0^a \rho^3 \, d\rho \\ = 2\pi \cdot \left[\frac{\sin^2\phi}{2}\right]_0^{\frac{\pi}{2}} \cdot \left[\frac{\rho^4}{4}\right]_0^a \\ = 2\pi \cdot \frac{1}{2} \cdot \frac{a^4}{4} = \frac{\pi a^4}{4}$$
Find $\bar{z}$
$$\bar{z} = \frac{\frac{\pi a^4}{4}}{\frac{2\pi a^3}{3}} = \frac{\pi a^4}{4} \cdot \frac{3}{2\pi a^3} = \frac{3a}{8}$$
ANSWER
Center of mass: $(0, 0, \frac{3a}{8})$
Problem 12 Question 16.140: Triple Integral Over Sphere/Hemisphere/Octant

Evaluate $\iiint_D (x^2 + y^2 + z^2)xyz \, dV$ over:

a) The sphere $D = \{x^2 + y^2 + z^2 \leq R^2\}$

b) The hemisphere $D = \{x^2 + y^2 + z^2 \leq R^2, z \geq 0\}$

c) The first octant $D = \{x \geq 0, y \geq 0, z \geq 0, x^2 + y^2 + z^2 \leq R^2\}$

Solution
Part (a): Full sphere
Symmetry argument

The integrand xyz is odd in each variable. The sphere is symmetric about each coordinate plane.

$$\text{Answer (a): } 0$$
Part (b): Upper hemisphere
Symmetry in x and y

Still symmetric about xz and yz planes, so integrand is still odd in x and y.

$$\text{Answer (b): } 0$$
Part (c): First octant - Full computation
Convert to spherical coordinates

$x = \rho \sin\phi \cos\theta$, $y = \rho \sin\phi \sin\theta$, $z = \rho \cos\phi$

$x^2 + y^2 + z^2 = \rho^2$, $dV = \rho^2 \sin\phi \, d\rho \, d\phi \, d\theta$

First octant: $0 \leq \rho \leq R$, $0 \leq \phi \leq \frac{\pi}{2}$, $0 \leq \theta \leq \frac{\pi}{2}$

Express the integrand
$$xyz = \rho^3 \sin^2\phi \cos\phi \sin\theta \cos\theta$$
$$(x^2+y^2+z^2)xyz = \rho^5 \sin^2\phi \cos\phi \sin\theta \cos\theta$$
Set up the integral
$$\int_0^{R} \int_0^{\pi/2} \int_0^{\pi/2} \rho^5 \sin^2\phi \cos\phi \sin\theta \cos\theta \cdot \rho^2 \sin\phi \, d\theta \, d\phi \, d\rho$$
$$= \int_0^{R} \rho^7 \, d\rho \cdot \int_0^{\pi/2} \sin^3\phi \cos\phi \, d\phi \cdot \int_0^{\pi/2} \sin\theta \cos\theta \, d\theta$$
Evaluate each integral
$$\int_0^{R} \rho^7 \, d\rho = \frac{R^8}{8}$$
$$\int_0^{\pi/2} \sin^3\phi \cos\phi \, d\phi = \left[\frac{\sin^4\phi}{4}\right]_0^{\pi/2} = \frac{1}{4}$$
$$\int_0^{\pi/2} \sin\theta \cos\theta \, d\theta = \left[\frac{\sin^2\theta}{2}\right]_0^{\pi/2} = \frac{1}{2}$$
Multiply
$$\frac{R^8}{8} \cdot \frac{1}{4} \cdot \frac{1}{2} = \frac{R^8}{64}$$
ANSWERS
(a) 0, (b) 0, (c) $\frac{R^8}{64}$

Solution

Key Concept: Cylindrical coordinates are ideal for regions with circular cross-sections and when $\sqrt{x^2 + y^2}$ appears in bounds or integrand.

Understand the cone region

The region C is defined by $\sqrt{x^2 + y^2} \leq z \leq 1$:

• Lower boundary: the cone $z = \sqrt{x^2 + y^2}$ (45° cone)

• Upper boundary: the plane $z = 1$

• This creates an ice cream cone shape with vertex at origin, capped at $z = 1$

Convert to cylindrical coordinates

In cylindrical: $x = r\cos\theta$, $y = r\sin\theta$, $z = z$

$\sqrt{x^2 + y^2} = r$, so the integrand becomes $1 + r$

$$\iiint_C (1 + r)\,dV = \int_0^{2\pi} \int_0^1 \int_r^1 (1 + r) \cdot r\,dz\,dr\,d\theta$$
Evaluate inner integral (z)

$(1 + r) \cdot r$ is constant with respect to $z$:

Expand and evaluate middle integral (r)
$$\int_0^1 r(1 - r^2)\,dr = \int_0^1 (r - r^3)\,dr = \left[\frac{r^2}{2} - \frac{r^4}{4}\right]_0^1 = \frac{1}{2} - \frac{1}{4} = \frac{1}{4}$$
Evaluate outer integral (θ)

Problem 14 Question 16.139: Integral Between Two Spheres

Evaluate $\iiint_S \frac{1}{(x^2 + y^2 + z^2)^{3/2}} \, dV$, where S is bounded by spheres $x^2 + y^2 + z^2 = a^2$ and $x^2 + y^2 + z^2 = b^2$ ($a > b > 0$)

Solution

Key Concept: The region between two concentric spheres (a "spherical shell") is perfectly suited for spherical coordinates. The integrand $1/r^3$ in 3D is significant in physics—it describes the potential energy density of a point charge.

Identify the region and convert to spherical

The region S is a spherical shell: all points between the inner sphere (radius $b$) and outer sphere (radius $a$).

In spherical coordinates:

$$\frac{1}{(x^2 + y^2 + z^2)^{3/2}} = \frac{1}{\rho^3}$$
$$dV = \rho^2 \sin \phi \, d\rho \, d\phi \, d\theta$$

Bounds: $b \leq \rho \leq a$, $0 \leq \phi \leq \pi$, $0 \leq \theta \leq 2\pi$

Set up and simplify the integrand

Combine the integrand with the volume element:

$$\frac{1}{\rho^3} \cdot \rho^2 \sin \phi = \rho^{-1} \sin \phi$$

The integral becomes:

$$\int_0^{2\pi} \int_0^{\pi} \int_b^a \rho^{-1} \sin \phi \, d\rho \, d\phi \, d\theta$$
Evaluate each integral separately (separable!)

Since the integrand factors as $\rho^{-1} \cdot \sin\phi \cdot 1$:

$$\int_b^a \rho^{-1} \, d\rho = [\ln\rho]_b^a = \ln a - \ln b = \ln\frac{a}{b}$$
$$\int_0^{\pi} \sin \phi \, d\phi = [-\cos\phi]_0^{\pi} = -(-1) - (-1) = 2$$
$$\int_0^{2\pi} d\theta = 2\pi$$
Multiply the results
$$\ln\frac{a}{b} \cdot 2 \cdot 2\pi = 4\pi \ln\frac{a}{b}$$

Physical interpretation: This integral arises when computing the "gravitational potential energy" or "electrostatic energy" stored in a spherical shell. The logarithmic dependence on the ratio $a/b$ (not the individual radii) reflects a scale invariance property.

ANSWER
$4\pi \ln(a/b)$

The density of a solid sphere of radius R is $(1 + d^3)^{-1}$ where d is distance from center. Find total mass.

Solution

Key Concept: Total mass = $\iiint \rho(x,y,z)\,dV$ where $\rho$ is density. When density depends only on distance from center, use spherical coordinates where that distance is simply $\rho$.

Identify the setup

The density is $\delta = \frac{1}{1 + d^3}$ where $d$ is distance from center.

In spherical coordinates, $d = \rho$ (the radial coordinate).

$$\text{density} = \frac{1}{1 + \rho^3}$$
$$dV = \rho^2 \sin \phi \, d\rho \, d\phi \, d\theta$$
Set up the mass integral
$$\text{Mass} = \iiint_{\text{sphere}} \frac{1}{1 + \rho^3}\,dV = \int_0^{2\pi} \int_0^{\pi} \int_0^R \frac{\rho^2}{1 + \rho^3} \sin \phi \, d\rho \, d\phi \, d\theta$$
Evaluate the ρ integral using substitution

Let $u = 1 + \rho^3$, then $du = 3\rho^2\,d\rho$, so $\rho^2\,d\rho = \frac{du}{3}$

When $\rho = 0$: $u = 1$. When $\rho = R$: $u = 1 + R^3$

$$\int_0^R \frac{\rho^2}{1 + \rho^3} \, d\rho = \frac{1}{3}\int_1^{1+R^3} \frac{du}{u} = \frac{1}{3}[\ln u]_1^{1+R^3} = \frac{1}{3} \ln(1 + R^3)$$
Evaluate the angular integrals
$$\int_0^{\pi} \sin \phi \, d\phi = [-\cos\phi]_0^{\pi} = 2$$
$$\int_0^{2\pi} d\theta = 2\pi$$
Combine all parts
$$\text{Mass} = \frac{1}{3} \ln(1 + R^3) \cdot 2 \cdot 2\pi = \frac{4\pi}{3} \ln(1 + R^3)$$

Note: This density decreases as you move away from center (since $1+\rho^3$ increases). The mass depends logarithmically on $R^3$, not linearly—the sphere gets "less dense" faster than it grows.

ANSWER
$\frac{4\pi}{3} \ln(1 + R^3)$

b) Find points where temperature equals average

Solution

Part (a): Average Temperature
Set up the average value formula

The cube $C = \{-1 \leq x,y,z \leq 1\}$ has side length 2.

$$V = 2^3 = 8$$

Temperature is $T = 32d^2 = 32(x^2 + y^2 + z^2)$.

$$T_{\text{avg}} = \frac{1}{V} \iiint_C T\,dV = \frac{1}{8} \iiint_C 32(x^2 + y^2 + z^2)\,dV$$
Use symmetry to simplify

By symmetry of the cube, $\iiint_C x^2\,dV = \iiint_C y^2\,dV = \iiint_C z^2\,dV$.

We need $T = 32$:

Problem 18 Question 16.125: Tetrahedron Volume

Find volume bounded by $\frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1$ and coordinate planes.

Solution

Key Concept: The plane $\frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1$ intercepts the axes at $(a,0,0)$, $(0,b,0)$, $(0,0,c)$. Together with the coordinate planes, this forms a tetrahedron.

Identify the region (tetrahedron)

Vertices: $(0,0,0)$, $(a,0,0)$, $(0,b,0)$, $(0,0,c)$

The region is the first-octant portion below the slanted plane.

Determine integration bounds

For each $x$, the $y$-bound comes from setting $z = 0$ in the plane equation:

$$\frac{x}{a} + \frac{y}{b} = 1 \implies y = b\left(1 - \frac{x}{a}\right)$$

For each $(x,y)$, the $z$-bound is:

$$z = c\left(1 - \frac{x}{a} - \frac{y}{b}\right)$$
$$0 \leq x \leq a, \quad 0 \leq y \leq b\left(1 - \frac{x}{a}\right), \quad 0 \leq z \leq c\left(1 - \frac{x}{a} - \frac{y}{b}\right)$$
Evaluate the z-integral
$$\int_0^{c(1-x/a-y/b)} dz = c\left(1 - \frac{x}{a} - \frac{y}{b}\right)$$
Evaluate the y-integral

Let $u = 1 - \frac{x}{a}$ (constant with respect to $y$). Then:

$$\int_0^{bu} c\left(u - \frac{y}{b}\right)dy = c\left[uy - \frac{y^2}{2b}\right]_0^{bu} = c\left(u \cdot bu - \frac{b^2u^2}{2b}\right) = c \cdot \frac{bu^2}{2}$$
Evaluate the x-integral

With $u = 1 - \frac{x}{a}$, we have $du = -\frac{1}{a}dx$, so $dx = -a\,du$. When $x = 0$: $u = 1$; when $x = a$: $u = 0$.

$$V = \int_0^a \frac{bc}{2}\left(1 - \frac{x}{a}\right)^2 dx = \frac{bc}{2} \int_1^0 u^2 \cdot (-a)\,du = \frac{abc}{2} \int_0^1 u^2\,du = \frac{abc}{2} \cdot \frac{1}{3} = \frac{abc}{6}$$

Classic Result: A tetrahedron with vertices at the origin and $(a,0,0)$, $(0,b,0)$, $(0,0,c)$ has volume $\frac{abc}{6}$. This is $\frac{1}{6}$ of the rectangular box $[0,a] \times [0,b] \times [0,c]$.

ANSWER
$\frac{abc}{6}$

Problem 20 Question 16.123: Cylinder Through Sphere (Napkin Ring)

A cylindrical hole of diameter 1 is bored through a sphere of radius 2. Axis passes through center. Find remaining volume.

Solution

Key Concept: This is the famous "napkin ring" problem! A cylinder drilled through a sphere leaves a ring-shaped solid. Remarkably, the volume depends only on the height of the ring, not on the sphere's radius.

Identify the geometry

Sphere: $x^2 + y^2 + z^2 = 4$ (radius $R = 2$)

Cylinder: $x^2 + y^2 = \frac{1}{4}$ (radius $a = \frac{1}{2}$, axis along $z$)

Find where cylinder meets sphere: $\frac{1}{4} + z^2 = 4 \implies z^2 = \frac{15}{4} \implies z = \pm\frac{\sqrt{15}}{2}$

Set up in cylindrical coordinates

The remaining solid is the sphere minus the cylinder. In cylindrical:

$$V = \int_0^{2\pi} \int_{-\sqrt{15}/2}^{\sqrt{15}/2} \int_{1/2}^{\sqrt{4-z^2}} r\,dr\,dz\,d\theta$$

Here $r$ ranges from $\frac{1}{2}$ (inner cylinder wall) to $\sqrt{4-z^2}$ (outer sphere surface).

Evaluate the r-integral
$$\int_{1/2}^{\sqrt{4-z^2}} r\,dr = \left[\frac{r^2}{2}\right]_{1/2}^{\sqrt{4-z^2}} = \frac{4-z^2}{2} - \frac{1}{8} = \frac{4-z^2}{2} - \frac{1}{8}$$
Evaluate the z-integral

Using symmetry ($z$ integrand is even):

$$\int_{-\sqrt{15}/2}^{\sqrt{15}/2} \left(\frac{4-z^2}{2} - \frac{1}{8}\right)dz = 2\int_0^{\sqrt{15}/2} \left(2 - \frac{z^2}{2} - \frac{1}{8}\right)dz$$
$$= 2\int_0^{\sqrt{15}/2} \left(\frac{15}{8} - \frac{z^2}{2}\right)dz = 2\left[\frac{15z}{8} - \frac{z^3}{6}\right]_0^{\sqrt{15}/2}$$
$$= 2\left(\frac{15 \cdot \sqrt{15}}{16} - \frac{(\sqrt{15})^3}{48}\right) = 2\left(\frac{15\sqrt{15}}{16} - \frac{15\sqrt{15}}{48}\right)$$
$$= 2 \cdot 15\sqrt{15}\left(\frac{1}{16} - \frac{1}{48}\right) = 30\sqrt{15} \cdot \frac{2}{48} = \frac{5\sqrt{15}}{4}$$
Multiply by θ-integral
$$V = 2\pi \cdot \frac{5\sqrt{15}}{4} = \frac{5\sqrt{15}\pi}{2}$$

Napkin Ring Theorem: If you drill a cylinder through a sphere such that the remaining ring has height $h$, the volume is always $\frac{\pi h^3}{6}$—independent of the original sphere's radius! Here $h = \sqrt{15}$, so $V = \frac{\pi \cdot 15^{3/2}}{6} = \frac{5\sqrt{15}\pi}{2}$.

ANSWER
$\frac{5\sqrt{15}\pi}{2}$

Compute the Jacobian of G(u,v) = (3ue^v, 7 + 3e^u)

Solution

Key Concept: The Jacobian determinant $\frac{\partial(x,y)}{\partial(u,v)} = \begin{vmatrix} \frac{\partial x}{\partial u} & \frac{\partial x}{\partial v} \\ \frac{\partial y}{\partial u} & \frac{\partial y}{\partial v} \end{vmatrix}$ measures how areas change under the transformation. It's the 2D analogue of the derivative.

Identify the transformation components

The transformation $G(u,v) = (x,y)$ gives:

$$x = 3ue^v, \quad y = 7 + 3e^u$$
Find all four partial derivatives
$$\frac{\partial x}{\partial u} = 3e^v \quad \text{(differentiate } 3ue^v \text{ w.r.t. } u)$$
$$\frac{\partial x}{\partial v} = 3ue^v \quad \text{(differentiate } 3ue^v \text{ w.r.t. } v)$$
$$\frac{\partial y}{\partial u} = 3e^u \quad \text{(differentiate } 7 + 3e^u \text{ w.r.t. } u)$$
$$\frac{\partial y}{\partial v} = 0 \quad \text{(} y \text{ has no } v)$$

$$\text{Jac} = \begin{vmatrix} 3e^v & 3ue^v \\ 3e^u & 0 \end{vmatrix} = (3e^v)(0) - (3ue^v)(3e^u)$$
$$= 0 - 9ue^v \cdot e^u = -9ue^{u+v}$$

Problem 24 Double Integral Using Change of Variables

Calculate $\iint_D (x + 4y) \, dx \, dy$ using $G(u,v) = (u - 2v, v)$

Answer: $96$

Solution

Key Concept: When using change of variables, transform three things: (1) the integrand, (2) the region, (3) multiply by $|\text{Jac}|$.

Find the Jacobian

From $G(u,v) = (u - 2v, v)$: $x = u - 2v$, $y = v$

$$\text{Jac} = \begin{vmatrix} \frac{\partial x}{\partial u} & \frac{\partial x}{\partial v} \\ \frac{\partial y}{\partial u} & \frac{\partial y}{\partial v} \end{vmatrix} = \begin{vmatrix} 1 & -2 \\ 0 & 1 \end{vmatrix} = (1)(1) - (-2)(0) = 1$$

Since $|\text{Jac}| = 1$, areas are preserved.

Transform the integrand

Express $x + 4y$ in terms of $u, v$:

$$x + 4y = (u - 2v) + 4v = u + 2v$$
Identify the new region

The problem specifies the $(u,v)$ region: $1 \leq v \leq 3$, $6 \leq u \leq 10$ (a rectangle).

Evaluate the transformed integral
$$\iint_D (x+4y)\,dA = \int_1^3 \int_6^{10} (u + 2v) \cdot 1\,du\,dv$$
$$= \int_1^3 \left[\frac{u^2}{2} + 2vu\right]_6^{10}\,dv = \int_1^3 \left[(50 + 20v) - (18 + 12v)\right]dv$$
$$= \int_1^3 (32 + 8v)\,dv = [32v + 4v^2]_1^3 = (96 + 36) - (32 + 4) = 132 - 36 = 96$$

$96$
Problem 25 Question 16.119: Change of Variables

Region B bounded by $xy = 1$, $xy = 3$, $x^2 - y^2 = 1$, $x^2 - y^2 = 4$. Evaluate $\iint_B (x^2 + y^2) \, dA$ using $u = x^2 - y^2$, $v = xy$

Solution

Key Concept: When the boundary curves are level sets of $u$ and $v$, the new region in $(u,v)$-space becomes a rectangle! This is why we choose coordinates that match the boundary curves.

Transform the region to $(u,v)$-space

The boundary curves become:

• $xy = 1 \to v = 1$, $xy = 3 \to v = 3$

• $x^2 - y^2 = 1 \to u = 1$, $x^2 - y^2 = 4 \to u = 4$

$$\text{New region: } 1 \leq u \leq 4, \quad 1 \leq v \leq 3 \quad \text{(a rectangle!)}$$
Find the Jacobian using the inverse

It's easier to compute $\frac{\partial(u,v)}{\partial(x,y)}$ and take the reciprocal:

$$\frac{\partial u}{\partial x} = 2x, \quad \frac{\partial u}{\partial y} = -2y, \quad \frac{\partial v}{\partial x} = y, \quad \frac{\partial v}{\partial y} = x$$
$$\frac{\partial(u,v)}{\partial(x,y)} = \begin{vmatrix} 2x & -2y \\ y & x \end{vmatrix} = 2x \cdot x - (-2y) \cdot y = 2x^2 + 2y^2 = 2(x^2 + y^2)$$

$$\left|\frac{\partial(x,y)}{\partial(u,v)}\right| = \frac{1}{2(x^2 + y^2)}$$

$3$

• The flat circular base: $x^2 + y^2 \leq 1$, $z = 0$

Given $\mathbf{E} = 8x\mathbf{i} + 8y\mathbf{j} + 8z\mathbf{k} = (8x, 8y, 8z)$:

Solution

Key Concept: A 2D vector field $\mathbf{F} = P\mathbf{i} + Q\mathbf{j}$ is conservative (i.e., the gradient of some potential function $f$) if and only if $\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$. This condition comes from the equality of mixed partial derivatives: if $\mathbf{F} = \nabla f$, then $P = \frac{\partial f}{\partial x}$ and $Q = \frac{\partial f}{\partial y}$, so $\frac{\partial P}{\partial y} = \frac{\partial^2 f}{\partial y \partial x} = \frac{\partial^2 f}{\partial x \partial y} = \frac{\partial Q}{\partial x}$.

Identify P and Q from the vector field

A vector field $\mathbf{F} = P\mathbf{i} + Q\mathbf{j}$ has components:

• $P$ = coefficient of $\mathbf{i}$ (the $x$-component)

• $Q$ = coefficient of $\mathbf{j}$ (the $y$-component)

From $\mathbf{F} = 3x\mathbf{i} + 7y\mathbf{j}$:

$$P = 3x, \quad Q = 7y$$
Test for conservative field (curl test)

For a 2D field to be conservative, we need $\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$:

$$\frac{\partial P}{\partial y} = \frac{\partial}{\partial y}(3x) = 0 \quad \text{(since } 3x \text{ has no } y \text{)}$$
$$\frac{\partial Q}{\partial x} = \frac{\partial}{\partial x}(7y) = 0 \quad \text{(since } 7y \text{ has no } x \text{)}$$

Since $0 = 0$ ✓, the field IS conservative and a potential function $f$ exists.

Set up the potential function conditions

If $\mathbf{F} = \nabla f$, then:

$$\frac{\partial f}{\partial x} = P = 3x$$
$$\frac{\partial f}{\partial y} = Q = 7y$$

We need to find $f(x,y)$ satisfying both conditions simultaneously.

Integrate P with respect to x

From $\frac{\partial f}{\partial x} = 3x$, integrate with respect to $x$ (treating $y$ as constant):

$$f(x,y) = \int 3x\,dx = \frac{3x^2}{2} + g(y)$$

Important: The "constant" of integration is actually a function $g(y)$ because when we differentiate $f$ with respect to $x$, any function of $y$ alone would vanish.

Find g(y) using the second condition

Now use $\frac{\partial f}{\partial y} = Q = 7y$ to find $g(y)$:

$$\frac{\partial f}{\partial y} = \frac{\partial}{\partial y}\left(\frac{3x^2}{2} + g(y)\right) = 0 + g'(y) = g'(y)$$

Setting this equal to $Q = 7y$:

$$g'(y) = 7y$$

Integrate with respect to $y$:

$$g(y) = \int 7y\,dy = \frac{7y^2}{2} + C$$
Write the complete potential function

Substitute $g(y)$ back into $f(x,y)$:

$$f(x,y) = \frac{3x^2}{2} + \frac{7y^2}{2} + C$$

The constant $C$ is arbitrary (any value works).

Verification

Always verify by checking $\nabla f = \mathbf{F}$:

$$\frac{\partial f}{\partial x} = \frac{\partial}{\partial x}\left(\frac{3x^2}{2} + \frac{7y^2}{2} + C\right) = 3x \quad \checkmark$$
$$\frac{\partial f}{\partial y} = \frac{\partial}{\partial y}\left(\frac{3x^2}{2} + \frac{7y^2}{2} + C\right) = 7y \quad \checkmark$$

Both match the original $\mathbf{F} = 3x\mathbf{i} + 7y\mathbf{j}$, confirming our answer.

Physical Interpretation: The potential function $f(x,y) = \frac{3x^2}{2} + \frac{7y^2}{2}$ represents an elliptic paraboloid. The vector field $\mathbf{F}$ points in the direction of steepest ascent on this surface, with magnitude equal to the slope.

ANSWER
$f(x,y) = \frac{3}{2}x^2 + \frac{7}{2}y^2 + C$
Problem 41 Conservative Field (b): F = 5xyi + 7xyj

Determine if F is gradient of f. If yes, find f.

Solution

Recall: A field $\mathbf{F} = P\mathbf{i} + Q\mathbf{j}$ is conservative iff $\frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x}$.

Identify P and Q

From $\mathbf{F} = 5xy\mathbf{i} + 7xy\mathbf{j}$:

$$P = 5xy, \quad Q = 7xy$$
Compute the partial derivatives
$$\frac{\partial P}{\partial y} = \frac{\partial}{\partial y}(5xy) = 5x$$
$$\frac{\partial Q}{\partial x} = \frac{\partial}{\partial x}(7xy) = 7y$$
Compare

For the field to be conservative, we need $5x = 7y$ for all $(x,y)$.

But $5x \neq 7y$ in general (only on the line $y = \frac{5x}{7}$).

Therefore, $\mathbf{F}$ is NOT conservative.

Consequence: Since $\mathbf{F}$ is not conservative, there is no potential function $f$ such that $\nabla f = \mathbf{F}$. Line integrals of $\mathbf{F}$ will depend on the path taken, not just the endpoints.

ANSWER
Not conservative (no potential function exists)
Problem 42 Conservative Field (c): F = (7x^2 + 7y^2)i + 14xyj

Determine if F is gradient of f. If yes, find f.

Solution
Test for Conservative

$P = 7x^2 + 7y^2$, $Q = 14xy$

$$\frac{\partial P}{\partial y} = 14y \\ \frac{\partial Q}{\partial x} = 14y \\ \frac{\partial P}{\partial y} = \frac{\partial Q}{\partial x} \quad \checkmark \text{ Conservative!}$$
Find the Potential Function f

Since $\frac{\partial f}{\partial x} = P = 7x^2 + 7y^2$:

$$f = \int (7x^2 + 7y^2) \, dx = \frac{7}{3}x^3 + 7xy^2 + g(y)$$
Determine g(y)

Take $\frac{\partial f}{\partial y}$ and set equal to $Q$:

$$\frac{\partial f}{\partial y} = 14xy + g'(y) = 14xy \\ g'(y) = 0 \implies g(y) = C$$
ANSWER
$f(x,y) = \frac{7}{3}x^3 + 7xy^2 + C$
