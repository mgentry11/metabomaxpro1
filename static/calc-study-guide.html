Essential Concepts

What is Multivariable Calculus?

In Calculus I and II, you worked with functions of one variable: y = f(x). In Calculus III, we extend these ideas to functions of multiple variables: z = f(x, y) or w = f(x, y, z). This lets us work with surfaces, volumes, and fields in 2D and 3D space.

The Four Types of Integrals

What We're Computing

  • Double Integrals ∬ - Sum over a 2D region (area, mass of a plate)
  • Triple Integrals ∭ - Sum over a 3D region (volume, mass of a solid)
  • Line Integrals ∫ - Sum along a curve (work done by a force)
  • Surface Integrals ∬ - Sum over a surface (flux through a membrane)

Coordinate Systems

Polar Coordinates (2D)
x = r cos θ, y = r sin θ, r² = x² + y² dA = r dr dθ ← Don't forget the r!
Cylindrical Coordinates (3D)
x = r cos θ, y = r sin θ, z = z dV = r dz dr dθ
Spherical Coordinates (3D)
x = ρ sin φ cos θ, y = ρ sin φ sin θ, z = ρ cos φ ρ² = x² + y² + z² dV = ρ² sin φ dρ dφ dθ
Use polar/cylindrical when you see circles (x² + y²). Use spherical when you see spheres (x² + y² + z²).

Problem 2 Iterated Integral of Cosine

Evaluate: ∫₋π⁰ ∫_{π/4}^{π/2} cos(2x + y) dy dx

Answer: 0

Solution
Inner integral (with respect to y)
∫_{π/4}^{π/2} cos(2x + y) dy = [sin(2x + y)]_{π/4}^{π/2} = sin(2x + π/2) - sin(2x + π/4) = cos(2x) - sin(2x + π/4)
Outer integral (with respect to x)
∫_{-π}^{0} [cos(2x) - sin(2x + π/4)] dx = [sin(2x)/2 + cos(2x + π/4)/2]_{-π}^{0}
Evaluate at bounds
At x = 0: sin(0)/2 + cos(π/4)/2 = 0 + (√2/2)/2 = √2/4 At x = -π: sin(-2π)/2 + cos(-2π + π/4)/2 = 0 + cos(π/4)/2 = √2/4 Result: √2/4 - √2/4 = 0
Why Zero?

The integral is zero because we're integrating periodic functions over complete periods, causing positive and negative contributions to cancel.

ANSWER
0

Problem 4 Question 16.138: Gaussian Over Lower Semicircle

Evaluate ∬_B e^{-x²-y²} dx dy, where B = {(x,y) | x² + y² ≤ 1 and y ≤ 0}

Solution
Convert to polar coordinates
x² + y² = r², so e^{-x²-y²} = e^{-r²} dA = r dr dθ Region: 0 ≤ r ≤ 1, π ≤ θ ≤ 2π (lower half where y ≤ 0)
Set up integral
∬_B e^{-r²} · r dr dθ = ∫_π^{2π} ∫_0^1 r·e^{-r²} dr dθ
Inner integral
∫_0^1 r·e^{-r²} dr Let u = -r², du = -2r dr, so r dr = -du/2 = -½[e^u]_{0}^{-1} = -½(e^{-1} - 1) = ½(1 - 1/e) = (e-1)/(2e)
Outer integral
∫_π^{2π} (e-1)/(2e) dθ = (e-1)/(2e) · π = π(e-1)/(2e)
ANSWER
π(1 - e⁻¹)/2 = π(e-1)/(2e)
Problem 5 Question 16.147b: Fourth Power Integral

Evaluate ∬_B (x⁴ + 2x²y² + y⁴) dx dy where B is the disk of radius 2 centered at origin in the first quadrant.

∫_0^2 r⁵ dr = [r⁶/6]_0^2 = 64/6 = 32/3 ∫_0^{π/2} (32/3) dθ = (32/3)(π/2) = 16π/3

Problem 6 Question 16.147a: Improper Double Integral

Evaluate: ∫_0^∞ ∫_0^y x·e^{-y³} dx dy

Solution
Inner integral (with respect to x)
∫_0^y x·e^{-y³} dx = e^{-y³} ∫_0^y x dx = e^{-y³} · [x²/2]_0^y = e^{-y³} · y²/2
Outer integral
∫_0^∞ (y²/2)·e^{-y³} dy Let u = y³, du = 3y² dy, so y² dy = du/3 = (1/2) · (1/3) ∫_0^∞ e^{-u} du = (1/6)[-e^{-u}]_0^∞ = (1/6)(0 - (-1)) = 1/6

Problem 7 Question 16.149: Integral Over All of R²

Evaluate ∬_{R²} f(x,y) dx dy where f(x,y) = 1/(1 + x² + y²)^{3/2}

Solution
Convert to polar
1 + x² + y² = 1 + r² f = 1/(1 + r²)^{3/2} dA = r dr dθ ∫_0^{2π} ∫_0^∞ r/(1 + r²)^{3/2} dr dθ

Problem 8 Integral with Square Root

Evaluate ∬_D 1/√(3xy) dA where D = [0,8] × [0,8]

Solution
Separate the integral
∬_D 1/√(3xy) dA = (1/√3) ∫_0^8 ∫_0^8 x^{-1/2} y^{-1/2} dx dy = (1/√3) ∫_0^8 x^{-1/2} dx · ∫_0^8 y^{-1/2} dy

Problem 9 Triple Integral with Symmetry

Evaluate ∭_W 9x dV for region W: x² + y² ≤ z ≤ 81

Answer: 0

Solution
Identify symmetry

The region W is symmetric about the yz-plane (if you replace x with -x, you get the same region).

The function f(x,y,z) = 9x is odd in x: f(-x,y,z) = -9x = -f(x,y,z)

Apply symmetry principle

When integrating an odd function over a symmetric region, the integral is zero.

The Symmetry Shortcut

For every point (x,y,z) contributing +9x, there's a point (-x,y,z) contributing -9x. They cancel!

ANSWER
0
Problem 10 Improper Integral Outside Sphere

Let D be the unbounded region x² + y² + z² ≥ 9. Evaluate: ∭_D 7/(x² + y² + z²)² dV

Answer: 28π/3

Solution
Convert to spherical
x² + y² + z² = ρ², so 7/(x² + y² + z²)² = 7/ρ⁴ dV = ρ² sin φ dρ dφ dθ Region: ρ ≥ 3, 0 ≤ φ ≤ π, 0 ≤ θ ≤ 2π
Set up integral
∫_0^{2π} ∫_0^π ∫_3^∞ (7/ρ⁴) · ρ² sin φ dρ dφ dθ = ∫_0^{2π} ∫_0^π ∫_3^∞ 7ρ^{-2} sin φ dρ dφ dθ
Evaluate ρ integral
∫_3^∞ 7ρ^{-2} dρ = 7[-ρ^{-1}]_3^∞ = 7(0 + 1/3) = 7/3
Evaluate φ integral
∫_0^π sin φ dφ = [-cos φ]_0^π = 1 + 1 = 2
Evaluate θ integral
∫_0^{2π} dθ = 2π
Combine
(7/3) × 2 × 2π = 28π/3
ANSWER
28π/3
Problem 11 Question 16.137: Center of Mass of Hemisphere

Find the center of mass of hemisphere V = {(x,y,z) | x² + y² + z² ≤ a², z ≥ 0} with constant density.

Solution
Use symmetry
x̄ = 0 (symmetric about yz-plane) ȳ = 0 (symmetric about xz-plane)
Calculate z̄ = ∭z dV / ∭dV

In spherical: z = ρ cos φ, dV = ρ² sin φ dρ dφ dθ

Region: 0 ≤ ρ ≤ a, 0 ≤ φ ≤ π/2, 0 ≤ θ ≤ 2π

Volume of hemisphere
∭ dV = (2/3)πa³
Calculate ∭z dV
∫_0^{2π} dθ · ∫_0^{π/2} cos φ sin φ dφ · ∫_0^a ρ³ dρ = 2π · [sin²φ/2]_0^{π/2} · [ρ⁴/4]_0^a = 2π · (1/2) · (a⁴/4) = πa⁴/4
Find z̄
z̄ = (πa⁴/4) / (2πa³/3) = (πa⁴/4) · (3/2πa³) = 3a/8

Problem 12 Question 16.140: Triple Integral Over Sphere/Hemisphere/Octant

b) The hemisphere D = {x² + y² + z² ≤ R², z ≥ 0}

Problem 13 Question 16.141: Cone-Shaped Region

Let C be the cone-shaped region {(x,y,z) | √(x² + y²) ≤ z ≤ 1}. Evaluate ∭_C (1 + √(x² + y²)) dV

Solution
Convert to cylindrical
√(x² + y²) = r, so region is: r ≤ z ≤ 1, 0 ≤ r ≤ 1 Integrand: 1 + r dV = r dz dr dθ
Set up integral
∫_0^{2π} ∫_0^1 ∫_r^1 (1 + r) · r dz dr dθ
Inner integral (z)
∫_r^1 dz = 1 - r
Middle integral (r)
∫_0^1 (1 + r) · r · (1 - r) dr = ∫_0^1 (r + r² - r² - r³) dr = ∫_0^1 (r - r³) dr = [r²/2 - r⁴/4]_0^1 = 1/2 - 1/4 = 1/4
Outer integral (θ)
∫_0^{2π} (1/4) dθ = π/2
ANSWER
π/2
Problem 14 Question 16.139: Integral Between Two Spheres

Evaluate ∭_S 1/(x² + y² + z²)^{3/2} dV where S is bounded by spheres x² + y² + z² = a² and x² + y² + z² = b² (a > b > 0)

∫_0^{2π} ∫_0^π ∫_b^a (1/ρ³) · ρ² sin φ dρ dφ dθ = ∫_0^{2π} ∫_0^π ∫_b^a ρ^{-1} sin φ dρ dφ dθ

Problem 15 Question 16.132: Mass of Solid with Variable Density

The density of a solid sphere of radius R is (1 + d³)^{-1} where d is distance from center. Find total mass.

Solution
Set up in spherical
d = ρ (distance from origin) density = 1/(1 + ρ³) dV = ρ² sin φ dρ dφ dθ Mass = ∫_0^{2π} ∫_0^π ∫_0^R ρ²/(1 + ρ³) sin φ dρ dφ dθ
Evaluate ρ integral
∫_0^R ρ²/(1 + ρ³) dρ Let u = 1 + ρ³, du = 3ρ² dρ = (1/3) ln(1 + R³)
Complete integration
Mass = (1/3) ln(1 + R³) · 2 · 2π = (4π/3) ln(1 + R³)
ANSWER
(4π/3) ln(1 + R³)
Problem 16 Question 16.133: Mass of Spherical Shell

Spherical shell with inner radius 1m, outer radius 2m, density 0.4d² g/cm³ where d is distance from center. Find total mass.

Solution
Set up integral
Mass = ∫_0^{2π} ∫_0^π ∫_1^2 0.4ρ² · ρ² sin φ dρ dφ dθ = 0.4 ∫_0^{2π} dθ · ∫_0^π sin φ dφ · ∫_1^2 ρ⁴ dρ
Evaluate
= 0.4 · 2π · 2 · [ρ⁵/5]_1^2 = 0.4 · 4π · (32 - 1)/5 = 0.4 · 4π · 31/5 = 9.92π
Note: Units need to be consistent (convert m to cm if density is in g/cm³)
ANSWER
9.92π (with consistent units)

Problem 19 Question 16.126: Volume Between Paraboloid and Cone

Find volume determined by z ≤ 6 - x² - y² and z ≥ √(x² + y²)

Solution
Find intersection
√(x² + y²) = 6 - x² - y² Let r = √(x² + y²): r = 6 - r² r² + r - 6 = 0 → (r+3)(r-2) = 0 → r = 2
Set up in cylindrical
V = ∫_0^{2π} ∫_0^2 ∫_r^{6-r²} r dz dr dθ

32π/3
Problem 20 Question 16.123: Cylinder Through Sphere (Napkin Ring)

Solution
Set up
Sphere: x² + y² + z² ≤ 4 Cylinder removed: x² + y² ≤ 1/4 In cylindrical: 1/2 ≤ r ≤ √(4-z²), -√(15/4) ≤ z ≤ √(15/4)

V_sphere = (4/3)π(8) = 32π/3 V_remaining = V_sphere - V_cylinder_inside

Change of Variables Formula
∬_R f(x,y) dA = ∬_{R*} f(x(u,v), y(u,v)) |∂(x,y)/∂(u,v)| du dv
Problem 21 Compute Jacobian

Compute the Jacobian of G(u,v) = (3ue^v, 7 + 3e^u)

Answer: -9ue^{u+v}

Solution
Find partial derivatives
x = 3ue^v, y = 7 + 3e^u ∂x/∂u = 3e^v, ∂x/∂v = 3ue^v ∂y/∂u = 3e^u, ∂y/∂v = 0
Calculate determinant
Jac = |3e^v 3ue^v| |3e^u 0 | = (3e^v)(0) - (3ue^v)(3e^u) = -9ue^{u+v}
ANSWER
-9ue^{u+v}

Problem 25 Question 16.119: Change of Variables

Green's Theorem
∮_C (P dx + Q dy) = ∬_D (∂Q/∂x - ∂P/∂y) dA
Problem 33 Green's Theorem Application

Use Green's Theorem to evaluate ∮_C F·dr where F(x,y) = (x², x²), C consists of y = x² and y = 4x for 0 ≤ x ≤ 4, counterclockwise.

Solution
Identify P and Q
P = x², Q = x² ∂Q/∂x = 2x, ∂P/∂y = 0
Apply Green's Theorem
∮_C F·dr = ∬_D (2x - 0) dA = ∬_D 2x dA
Evaluate double integral
∫_0^4 ∫_{x²}^{4x} 2x dy dx = ∫_0^4 2x(4x - x²) dx = ∫_0^4 (8x² - 2x³) dx = [8x³/3 - x⁴/2]_0^4 = 512/3 - 128 = 128/3
ANSWER
128/3

CONSERVATIVE FIELDS

Test for Conservative (2D)
F = ⟨P, Q⟩ is conservative ⟺ ∂P/∂y = ∂Q/∂x
Problem 40 Conservative Field (a): F = 3xi + 7yj

Determine if F is gradient of f. If yes, find f.

Solution
Test
∂P/∂y = 0, ∂Q/∂x = 0 ✓ Equal!
Find f
∂f/∂x = 3x → f = (3/2)x² + g(y) ∂f/∂y = g'(y) = 7y → g(y) = (7/2)y² f(x,y) = (3/2)x² + (7/2)y² + C
ANSWER
f(x,y) = (3/2)x² + (7/2)y² + C
Problem 41 Conservative Field (b): F = 5xyi + 7xyj

Determine if F is gradient of f. If yes, find f.

Solution
Test
∂P/∂y = 5x, ∂Q/∂x = 7y 5x ≠ 7y in general ✗ NOT conservative
ANSWER
DNE (Not conservative)
Problem 42 Conservative Field (c): F = (7x² + 7y²)i + 14xyj

Determine if F is gradient of f. If yes, find f.

Solution
Test for Conservative

P = 7x² + 7y², Q = 14xy

∂P/∂y = 14y ∂Q/∂x = 14y ∂P/∂y = ∂Q/∂x ✓ Conservative!
Find the Potential Function f

Since ∂f/∂x = P = 7x² + 7y²:

f = ∫(7x² + 7y²)dx = (7/3)x³ + 7xy² + g(y)
Determine g(y)

Take ∂f/∂y and set equal to Q:

∂f/∂y = 14xy + g'(y) = 14xy g'(y) = 0 → g(y) = C
ANSWER
f(x,y) = (7/3)x³ + 7xy² + C
