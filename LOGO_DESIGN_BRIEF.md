# MetaboMaxPro Logo Design Brief

## Brand Overview
**Company Name:** MetaboMaxPro
**Industry:** Health Tech / Metabolic Testing / AI Analytics
**Mission:** Make metabolic test insights accessible and affordable
**Target Audience:** Athletes, biohackers, longevity enthusiasts, testing facilities

---

## Logo Requirements

### Primary Use Cases:
1. **Website** (header, favicon)
2. **LinkedIn** (company page profile pic - 300x300px)
3. **Social Media** (Twitter, Instagram, Facebook)
4. **Reports** (PDF headers, watermarks)
5. **Marketing Materials** (business cards, email signatures)
6. **White-Label** (needs to be versatile enough for partners to co-brand)

### Technical Specs:
- **Formats needed:** SVG (vector), PNG (transparent background), JPG
- **Sizes:**
  - Square version for social (300x300, 512x512, 1024x1024)
  - Horizontal version for website header
  - Favicon (16x16, 32x32, 64x64)
- **Colors:** Should work in full color, monochrome, and reversed (white on dark)

## Deliverables Checklist

When you receive your logo, ensure you get:

**File Formats:**
- [ ] SVG (vector - scalable)
- [ ] PNG (transparent background - multiple sizes)
- [ ] JPG (white background)
- [ ] AI or EPS (editable vector)

**Variations:**
- [ ] Full color horizontal
- [ ] Full color stacked (square)
- [ ] Icon only (no text)
- [ ] Monochrome (black)
- [ ] Reversed (white)
- [ ] Favicon sizes (16x16, 32x32, 64x64)

**Sizes:**
- [ ] 4000px wide (print quality)
- [ ] 1024x1024 (social media)
- [ ] 512x512 (social media)
- [ ] 300x300 (LinkedIn)
- [ ] 200x200 (profile pics)

**Documentation:**
- [ ] Color codes (HEX, RGB, CMYK)
- [ ] Font names and weights
- [ ] Usage guidelines
- [ ] Minimum size specifications
